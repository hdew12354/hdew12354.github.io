Drive-Scan Tool

'"Nothing good Comes from Boredom" yeah no fucker i beg to differ then beg'

So for this project, i was trying to come up with a solution to a problem one of my buddies had with their computer, as it wasn't booting, and i wanted to make a tool to see if there were any recoverable files on the drice or if it had to be wiped.

I decided to create this tool using WPF and C# because i am skilled in c# and i was learning WPF in college, therefore it seemed a brilliant opportunity to learn and get some practical knowledge in WPF.

to create this tool i needed a few main things, i needed

  • Visual Studio
  • a USB thumbdrive for testing

when i was making this i faced a few major challenges, one of which being looking through folders and finding files, as i was not very well versed in working with files from the outset but by the end of the project i was proficient in creating, saving and modifying folders and files.

in creating this tool, i demonstrated skills in organisation and willingness to learn, adapt and improvise among other things.
